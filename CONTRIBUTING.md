# Contributing to Morning Surge Short Strategy

Contributions are welcome! Here's how:

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-idea`
3. Make your changes and commit: `git commit -m "feat: your idea"`
4. Push and open a Pull Request

## Good First Issues
- Add real NSE data loader (using nsepy or jugaad-trader)
- Add Telegram alert on signal
- Add options strategy (buy puts instead of shorting)
- Improve backtesting with walk-forward validation
